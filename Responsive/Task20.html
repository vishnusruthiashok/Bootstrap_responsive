What I do

Static Websites

A Static Website consists of a series of HTML files, each one representing a physical page of a website. On static sites, each page is a separate HTML file. When you visit the homepage, you are viewing only the actual homepage file.

Responsive Web design

Responsive web design is about creating web pages that look good on all devices! A responsive web design will automatically adjust for different screen sizes and viewports.

Customisable Layouts

Customisable sites are templates with set layouts. You may be able to change photos and colors, but your site will look similkar to many other sites due to template.
